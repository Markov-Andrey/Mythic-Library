## Installation

Config Open Server:
- Apache 2.4
- PHP 8.1
- MySQL 8.0

## Deployment

- new DB in PhpMyAdmin - mythic-library
-  .env copy (you can set it yourself or ask the author)
- composer install
- npm run dev
- php artisan storage:link
- php artisan migrate
- clone DB
